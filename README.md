# tsdd-control
* NODE_ENV node环境变量
* APP_ENV 应用环境变量
  - dev 开发环境
  - pord 生产环境
* IS_CONFIG 是否添加配置文件

## 安装

```sh
pnpm install
```

## 本地开发

``` sh
pnpm dev
```

## 编译

``` sh
pnpm build
```

## 配置文件编译

``` sh
pnpm build:config
```

## 本地预览
> 先执行编译再执行该命令

``` sh
pnpm serve
```
## eslint检测

``` sh
pnpm lint
```

## eslint修复

``` sh
pnpm lint:fix
```
## prettier检测

```sh
pnpm format
```

## Git commit

``` sh
pnpm cz
```
